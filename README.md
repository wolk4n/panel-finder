# Panel Bulucu
Bu program girilen hedef sitenin admin panelini deneme/yanılma (brute-force) yöntemi ile bulur. Çıkan tüm sonuçlar doğru olmayabilir. Çünkü program sayfanın içeriğine bakmaz sadece sayfaya girip girmediğine bakar yani HTTP kodu 200 olup olmadığına bakar.

### Programın İndirilmesi ve Kullanımı
`git clone https://github.com/wolkann/panel-finder.git`<br>
`cd panel-finder`<br>
`python3 panel-finder.py` Yazarak programı çalıştırabilir ve gelen ekranda hedef sitenizi <strong>"https://google.com/"</strong> şeklinde yazarak programı başlatabilirsiniz.<br>
